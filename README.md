# 🔍 Regex Lab

An interactive web tool to **build, generate, and compare regular expressions**.

---

## ✨ Features

* 🎯 Regex Builder with symbol palette
* 🔤 Generate matching strings
* ⚖️ Regex equivalence checker
* 🎨 Clean, modern UI inspired by real tools

---
